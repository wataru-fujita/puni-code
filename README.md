# ぷにコード ～かわいいスライムにプログラミング！～ 🎮🩷

かわいいスライムの「ぷにちゃん」をプログラミングブロックで動かしてゴールを目指す、小学生向けのプログラミング学習パズルゲームです！
GIGAスクール構想などで配られている、小学校のタブレット学習でも遊びやすいように**「横画面」**でのプレイを前提として設計しています。

👉 **[今すぐ遊んでみる！（公開URL）](https://wataru-fujita.github.io/puni-code/)**

---

## 🌟 特徴
- **直感的な操作**: ドラッグ＆ドロップでブロックを組み合わせて、楽しくプログラミング的思考を学べます。
- **タブレット最適化**: 小学生の娘たちが学校のタブレットで学習している姿を見て、横画面で快適に操作できるように画面レイアウトを工夫しました。
- **インタラクティブな演出**: 画面のあちこちをタップ・クリックすると、キラキラしたパーティクル（粒子）のエフェクトが飛び散ります！
- **ひみつのダークモード🌙**: タイトル画面の「ぷにちゃん」をたくさんクリックして、可愛いパーティクルを降らせてみてください。よく見ると、このアプリの裏本質である「ダークモード」へ繋がる秘密の導線が隠されているかも……？
