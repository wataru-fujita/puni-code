# ぷにコード ～かわいいスライムにプログラミング！～ 🎮🩷

かわいいスライムの「ぷにちゃん」をプログラミングブロックで動かしてゴールを目指す、小学生向けのプログラミング学習パズルゲームです。

👉 **[今すぐ遊んでみる！](https://wataru-fujita.github.io/puni-code/)**

---

## 🌟 特徴と「お父さんのこだわり」

*   **直感的な操作**: ドラッグ＆ドロップでブロックを組み合わせて、楽しくプログラミング的思考を学べます。
*   **タブレット最適化（横画面前提）**: GIGAスクール構想で小学校に一人一台配られているタブレット端末を意識し、横画面でのプレイを前提に設計しました。子どもたちが自分専用の画面で、自分のペースで楽しめることを大切にしています。
*   **インタラクティブな演出**: 画面をタップすると、キラキラした星やハートのパーティクル（粒子エフェクト）が飛び散ります！
*   **ひみつのダークモード 🌙**: タイトル画面の「ぷにちゃん」をタップするとパーティクルが降ってきます。でも、よく見てください。その中に**「ぷにちゃんじゃない絵文字」**が紛れていませんか？ 😎🌈 ――それが、ダークモードへと繋がる秘密の入り口です。ファミコンの攻略本の裏技コーナーのようなワクワク感を仕込みました。

---

## 🚀 開発ストーリー

このプロジェクトは、**HTML・CSS・JavaScriptの名前しか知らなかった素人**が、AIコーディングエージェント **「Antigravity」** と二人三脚で作り上げた、バイブコーディングの結晶です。

もともとは2025年末にAntigravityを使い始めた頃に作りかけて、そのまま眠っていたプロジェクトでした。再び火がついたきっかけは、離れて暮らす2人の娘たちに「タブレットをプレゼントしたい」と思ったことでした。タブレットがただの遊び道具ではなく、学びの入り口になるなら――そう考えた瞬間、眠っていたプロジェクトに強い動機が生まれ、開発から `git push` による公開まで一気に駆け抜けました。

数日、数週間と悩んでいる間に、子どもたちはどんどん成長してしまいます。「いつか作る」ではなく、想いが一番熱いうちにAIと一緒に形にして、世界に `push` する。完璧じゃなくてもいいから、まず届けることの大切さを、この開発を通じて学びました。

もしかすると、娘たちの成長のほうが速くて、すでに対象年齢を少し過ぎてしまっているかもしれません（時すでにお寿司、です）。それでも、「お父さん、こういうの作ってたんだ」と笑ってくれたら、それだけで十分だと思っています。

開発の詳しい経緯は、noteの記事に書きました。よければ読んでみてください。

📝 **[AIと一緒に「push」した夜、娘たちは少し先へ進んでいた｜河豚田](https://note.com/wataru_fujita)**
