# ぷにコード ～かわいいスライムにプログラミング！～ 🎮🩷

かわいいスライムの「ぷにちゃん」をプログラミングブロックで動かしてゴールを目指す、小学生向けのプログラミング学習パズルゲームです。

👉 **[今すぐ遊んでみる！（公開URL）](https://wataru-fujita.github.io/puni-code/)**

---

## 🌟 特徴と「お父さんのこだわり」
*   **直感的な操作**: ドラッグ＆ドロップでブロックを組み合わせて、楽しくプログラミング的思考を学べます。
*   **タブレット最適化（横画面前提）**: 娘たちが学校のタブレットで学習している姿を見て、お下がりや交代ではなく「自分専用の画面」で快適に操作できるように画面レイアウトを設計しました。
*   **インタラクティブな演出**: 画面をタップすると、キラキラした星やハートのパーティクル（粒子）のエフェクトが飛び散ります！
*   **ひみつのダークモード（裏技）🌙**: 
    タイトル画面の「ぷにちゃん」をタップするとパーティクルが降りますが、実はそこに紛れている**「ぷにちゃんじゃない絵文字（サングラスの顔 😎 や 虹 🌈 など）」**が、裏本質である「ダークモード」へと繋がる秘密の入り口になっています。ファミコンの攻略本の裏技コーナーのようなワクワク感を仕込みました。

## 🚀 開発ストーリー（思いの鮮度をAIで加速する）
このプロジェクトは、**「HTML、CSS、JSの名前しか知らなかった素人」**が、AIコーディングエージェント **「Antigravity」** と二人三脚で、**わずか1日**で完成から公開までやり遂げたバイブコーディングの結晶です。

もともとは2025年末に作りかけて放置されていたプロジェクトでした。しかし、離れて暮らす娘たちへのプレゼントをきっかけに、「思いの鮮度があるうちに届けたい」という強い動機が生まれ、一気に火がつきました。

数日、数週間と悩んでいる間に、子どもたちはどんどん成長してしまいます。
「いつか作る」ではなく、AIと一緒に今すぐ形にして、世界に `push` する。完璧じゃなくてもいいから、想いが一番熱いうちに届けることの大切さを、この開発を通じて学びました。

もしかすると、娘たちの成長のスピードのほうが速くて、すでに対象年齢を少し過ぎてしまっているかもしれません（時すでにお寿司、です）。それでも、「お父さん、こういうの作ってたんだ」と笑ってくれたら、それだけで十分だと思っています。
